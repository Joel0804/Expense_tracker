# Expense tracker
A simple command-line Expense tracker built with Python and OOP.

## Features
- Add, update, delete, and sum contacts
- Show all expenses
- Case-insensitive search
- Object-Oriented Programming structure
- Json to load and save data 
- API integration

## How to Run
1. Make sure Python is installed
2. Clone the repo:
   git clone https://github.com/your-username/Expense_tracker.git
3. Run the program:
   python main.py
